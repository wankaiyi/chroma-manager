# 介绍
chroma-manager是向量数据库chroma的可视化工具，方便查看 database、collections、documents 和 embeddings，并提供直观的操作界面。
# 使用方法
1. clone项目
2. Windows：运行start.bat；Linux、MacOS：运行start.sh。（可在脚本中修改运行的端口）

**注意使用jdk17，nodejs遇到问题试试v18.20.0**
